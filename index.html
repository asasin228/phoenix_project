What is it?

This is a Linux distribution designed for beginners with a beautiful working environment every morning budgie!Phoenix is very similar to Solus from Budgie.Phoenix is fully compatible with the openSUSE and SLE 12.Phoenix uses a package Manager Zypper and RPM package format

I have to pay for it?

You do not pay a single penny! This distribution is a completely free and open source No advertising and surveillance
